Massive Type

Big Shoulders Display at absurd sizes. It fills the frame, gets cropped, bleeds off edges. Type IS the design. Everything else supports it.

Organic Clusters

Soft radial gradient circles. They overlap, blur, breathe. Like thermal imaging of a dancefloor. Always radial-gradient to transparent. Never hard edges.

Raw Geometry

Thin rectangular blocks at low opacity slice through compositions. No rounded corners on UI. Sharp edges contrast the organic bloom. Grid borders at 1px, barely visible.

Data Residue

Monospaced coordinate labels at 15% opacity. They suggest a generative system underneath. Metadata from an imaginary algorithm that mapped the festival's energy.

07GUIDELINES

Rules

Do

  • Fill the frame with type. Let it get cropped. Let it bleed.
  • Use max 2–3 bloom circles per composition
  • Keep coordinate labels near invisible (15% opacity max)
  • Sharp edges on all UI. No border-radius.
  • 1px borders, barely there. Let structure be implied.
  • Grain texture on everything. Heavier = rawer.
  • Leave massive negative space. Let it breathe.
  • mix-blend-mode: screen on bloom circles

Don't

  • Use white backgrounds. Ever.
  • Put bloom accent colors on text. Background only.
  • Add more than 3 elements to a social post. Less is more.
  • Use drop shadows. Use translate + subtle border instead.
  • Make it decorative. Every element earns its place.
  • Use rounded buttons or pill shapes.
  • Shrink the type. If it feels too big, it's right.
  • Use Inter, Roboto, Arial. Big Shoulders + Work Sans only.
